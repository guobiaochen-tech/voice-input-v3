# Voice Input

macOS 菜单栏语音输入工具。按住快捷键说话，松开后文字自动输入到当前光标位置。支持语音转文字、润色、翻译。

## 功能

### 语音识别
- **按住说话**：按住快捷键开始录音，松开自动识别并输入文字
- **双引擎识别**：云端识别（阿里云 DashScope）和本地离线识别
- **自动降级**：云端连接失败时，若已下载本地模型，自动切换到本地识别
- **实时预览**：录音时屏幕右上角浮动指示器显示实时识别中的文字

### 文字处理（润色 / 翻译）
- **处理模式**：关闭（纯转文字）/ 润色 / 翻译，三种模式可切换
- **回复风格**：润色模式下可选高情商、幽默、暖男、暧昧拉扯、情场高手等多种语气（提示词以 md 文件存放在 `~/.voice-input/prompts/styles/`，可自行增删改，下拉框自动识别）
- **翻译语言**：翻译模式下可选英文、日文、韩文、德文等（同样以 md 文件管理，位于 `prompts/translate/`）

### 模型管理
- **语音模型**：云端服务商切换、API Key 配置、连接测速；本地模型下载 / 切换 / 删除、测速
- **润色模型**：内置 DeepSeek、Kimi、MiniMax、通义、火山豆包等预设，选服务商自动填地址和模型；支持拉取该 Key 下可用模型列表；响应速度测速

### 其他
- **快捷键自定义**：默认右 Command，可在设置中录制任意键或组合键
- **开机自启**：菜单栏可切换
- **提示音 / 保存录音 / 中英文自动加空格**等选项

## 安装

### 方式一：下载 DMG（推荐）

1. 到本仓库的 [Releases](../../releases) 页面下载最新 DMG
2. 打开 DMG，将 VoiceInput 拖入「应用程序」文件夹
3. 首次打开：因为 app 是本地签名（非苹果开发者证书），双击会被系统拦截，请**右键点击 app → 打开** → 在弹窗里再点「打开」放行（只需第一次）
4. 首次启动需授权：
   - **辅助功能权限**：用于模拟键盘输入，把文字输入到当前应用
   - **麦克风权限**：用于录音

### 方式二：自行编译

```bash
git clone <仓库地址>
cd voice-input-v3
bash build.sh
```

`build.sh` 会自动下载 sherpa-onnx 动态库和头文件，编译并打包成 `VoiceInput.app` 和 DMG。

## 使用

1. 启动后菜单栏出现图标
2. 按住快捷键（默认右 Command）开始说话，录音过程中可看到实时识别文字
3. 松开快捷键，等待识别完成
4. 文字（经润色或翻译，若开启）自动输入到当前光标位置
5. 录音中按 ESC 可取消

### 云端识别

需要阿里云 DashScope 的 API Key，在「设置 → 模型 → 语音模型」填入即可。

### 本地识别

无需联网。首次使用在「设置 → 模型 → 语音模型」选择本地识别，下载模型（首次约几百 MB），完成后即可离线使用。

## 配置

所有配置存放在 `~/.voice-input/`：

- `config.json`：非敏感设置（引擎、模式、模型名、快捷键等），**不存 API Key**
- `.env`：API Key（文件权限 600，仅本人可读）
- `prompts/styles/`、`prompts/translate/`：润色风格和翻译语言的提示词 md，可自行编辑
- `models/`：本地识别模型

设置改动后点「确认」保存。也可以直接设置环境变量 `ASR_API_KEY` / `POLISH_API_KEY`（或加 `VOICE_INPUT_` 前缀），优先级高于 `.env`。

## 系统要求

- macOS 13.0+
- 仅支持 Apple Silicon（M1/M2/M3 及以上）
- 本地识别首次使用需联网下载模型
