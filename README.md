# Voice Input

macOS 菜单栏语音输入工具。按住快捷键说话，松开后文字自动输入到当前应用。

## 功能

- **按住说话**：按住快捷键开始录音，松开自动识别并输入文字
- **双引擎识别**：支持云端识别和本地离线识别
- **自动切换**：云端失败自动切本地，本地失败自动切云端
- **浮动指示器**：录音时屏幕右上角显示实时状态，能看到识别中的文字
- **自定义快捷键**：默认右 Command 键，可在设置中修改为任意组合键

## 安装

1. 下载最新 DMG 文件
2. 打开 DMG，将 VoiceInput 拖入「应用程序」文件夹
3. 首次打开需要授权：
   - **辅助功能权限**：用于模拟键盘输入
   - **麦克风权限**：用于录音

## 使用

1. 启动后菜单栏出现图标
2. 按住快捷键（默认右 Command）开始说话
3. 松开快捷键，等待识别完成
4. 文字自动输入到当前光标位置
5. 按 ESC 可取消录音

### 云端识别

需要API Key，在设置中填入即可。

### 本地识别

无需联网,首次使用需在设置中下载模型,下载完成后即可离线使用。

## 设置

点击菜单栏图标 → 设置：

- **快捷键**：自定义录音快捷键
- **识别引擎**：选择云端或本地
- **本地模型**：下载/切换本地识别模型
- **其他选项**：中英文自动加空格等

## 系统要求

- macOS 13.0+
- Apple Silicon / Intel
